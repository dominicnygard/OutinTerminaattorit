# OutinTerminaattorit

Sovellus, johon käyttäjä voi lisätä LaTeX-lähdeviitteitä tietoineen ja tarkastella niitä.

Tilanne tällä hetkellä:
Käyttäjä pystyy lisätä viitteitä kaikista kategorioista ja hakea viitteitä niiden nimen perusteella.

Sovellus otetaan käyttöön kloonamalla projekti ja sitten ajaa  
`poetry install`  
projektin juuressa.  
Tämän jälkeen ajetaan  
`poetry run python src/index.py`  
komennon ajamisen jälkeen sivu löytyy osoitteesta http://localhost:5001
